# Titanic Project 🚢
This project analyzes the Titanic dataset with a focus on Exploratory Data Analysis (EDA), Feature Engineering, and Fare Prediction using regression models.

# 📂 Project Structure

- data/ → Titanic dataset (CSV files)
- Graphs(EDA)/ → saved plots and visualizations
- EDA.py → script for exploratory data analysis
- Feature.py → script for feature engineering
- models/ → trained model and scaler saved with joblib

# 📊 Exploratory Data Analysis

- Linear Regression applied for Fare prediction
- Performance metrics:
  - MAE → 20.36
  - MSE → 1115.75
- Model + Scaler stored using joblib

# 🌐 Streamlit UI
Interactive app built with Streamlit:
- Sidebar inputs for passenger attributes
- Predict button with green success output
- EDA graphs displayed alongside predictions
